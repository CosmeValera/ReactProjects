# Monorepo
All code for multiple projects is stored in a single repository.
Options for monorepos are: NX, and Turborepo.

## NX
```bash
npx create-nx-workspace@latest nxapp
```

## Turborepo
```bash
npx create-turbo@latest turboapp
```
